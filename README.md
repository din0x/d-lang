# d-lang
Dumb programming language


## Installation
 
1. Clone the repo
   ```sh
   git clone https://github.com/din0x/d-lang.git
   cd d-lang
   ```
   
2. Build with cargo
   ```sh
   cargo build --release
   ```
   For debug builds use
   ```sh
   cargo build
   ```

## Usage
For now use can only use cli (command line interpreter) by running the executable.
